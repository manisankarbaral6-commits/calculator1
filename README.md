# simple Calculator in C
This projectnis asimple Calculator Program implemented in the C programming language.
It allow users to perform basic arithmetic operations such as addition, substraction,multiplication and division(+,-,*,/).
The program takes two numbers and an operator as input and display the computed result.
This projec is beginner-friendly and helps to understand the fundamental of C programming such as input/output,conditional statements,and operators.
